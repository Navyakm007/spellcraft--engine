# 🪄 Harry Potter Theme Assets Setup

Your spell generator is ready for HP theming! Follow these steps to add images and music.

---

## 📸 Step 1 — Add Background Image

### Download a Hogwarts/Magical Background

Choose one (free options):

1. **Pixabay** (best quality) → Search "Hogwarts castle" or "magical castle"
   - https://pixabay.com/search/hogwarts/

2. **Unsplash** → Search "fantasy castle" or "magical background"
   - https://unsplash.com/search/fantasy

3. **Wallpaper Sites** → PexelsHD, WallpaperCave (search "Hogwarts")

### Add to Project

1. Download an image (JPG or PNG)
2. Save it in your project folder as: **`bg.jpg`**
   
```
spell-generator/
├── index.html
├── style.css
├── script.js
├── README.md
└── bg.jpg          ← Save here
```

3. Refresh your browser — the background appears instantly!

---

## 🎵 Step 2 — Add Theme Music

### Download Harry Potter Theme (Royalty-Free)

Use one of these:

1. **YouTube Audio Library** (free, no copyright)
   - Search "Harry Potter" → Download the instrumental version
   - https://www.youtube.com/audiolibrary

2. **Free Music Archive** → Search "Harry Potter theme"
   - https://freemusicarchive.org/

3. **Incompetech** → Search "wizard" or "magic theme"
   - https://incompetech.com/

### Add to Project

1. Download audio file as MP3
2. Save in project folder as: **`theme.mp3`**

```
spell-generator/
├── index.html
├── style.css
├── script.js
├── README.md
├── bg.jpg
└── theme.mp3       ← Save here
```

3. Refresh browser — music starts on first spell cast!

---

## 🔊 Optional: Add Spell Sound Effect

If you want spell-casting sound:

1. Search "magic spell sound" on:
   - Freesound.org
   - Zapsplat.com

2. Save as: **`spell.mp3`**

3. Already configured in your code!

## 💡 Pro Tips

- Use **PNG** images for transparency effects
- Choose music that's **1-2 minutes long** for looping
- Spell sounds should be **short** (0.5-1 second)
- Test on different browsers — music autoplay varies
